# Ejemplos de productos y servicios que utilizan Machine Learning

## Netflix

### Problema que resuelve con Machine Learning
Recomendar películas y series que se adapten a los gustos de cada usuario.

### Tipo de Machine Learning
**Aprendizaje supervisado.**

El aprendizaje supervisado es un tipo de machine learning en el que el modelo aprende a partir de datos que ya están etiquetados o clasificados. Es decir, el sistema recibe ejemplos donde ya se conoce el resultado correcto y aprende a predecir resultados similares en el futuro. En este caso, aprende a partir del comportamiento previo de los usuarios.

### Datos que probablemente necesita
- Historial de contenido visto  
- Tiempo de visualización  
- Preferencias del usuario  
- Interacciones con la plataforma  

---

## Spotify

### Problema que resuelve con Machine Learning
Sugerir música y playlists personalizadas según los gustos del usuario.

### Tipo de Machine Learning
**Aprendizaje supervisado y aprendizaje no supervisado.**

El aprendizaje no supervisado es un tipo de machine learning en el que el sistema analiza datos sin etiquetas previas y busca patrones o agrupaciones por sí mismo. Esto permite encontrar relaciones entre canciones con características similares.

También utiliza aprendizaje supervisado para aprender de los hábitos de escucha de los usuarios.

### Datos que probablemente necesita
- Historial de canciones escuchadas  
- Playlists creadas  
- Canciones omitidas  
- Duración de escucha  
- Preferencias musicales  

---

## Uber

### Problema que resuelve con Machine Learning
Predecir tiempos de llegada de los conductores y calcular precios de los viajes según la demanda.

### Tipo de Machine Learning
**Aprendizaje supervisado.**

El sistema utiliza datos históricos de viajes anteriores para entrenar modelos que puedan predecir cuánto tiempo tardará un conductor en llegar o cuánto costará un viaje en determinadas condiciones.

### Datos que probablemente necesita
- Ubicación de usuarios y conductores  
- Distancia del viaje  
- Tráfico  
- Hora del día  
- Historial de viajes  

---

## Amazon

### Problema que resuelve con Machine Learning
Recomendar productos que puedan interesar a los usuarios según su comportamiento de compra.

### Tipo de Machine Learning
**Aprendizaje supervisado.**

El sistema aprende a partir de los datos de compras y búsquedas realizadas anteriormente para predecir qué productos podrían interesar al usuario.

### Datos que probablemente necesita
- Historial de compras  
- Productos buscados  
- Productos visualizados  
- Interacciones con la plataforma  

---

## Mercado Libre

### Problema que resuelve con Machine Learning
Mejorar las recomendaciones de productos y detectar posibles fraudes en las transacciones.

### Tipo de Machine Learning
**Aprendizaje supervisado.**

El sistema analiza grandes cantidades de transacciones pasadas que fueron clasificadas como normales o fraudulentas para aprender a detectar comportamientos sospechosos.

### Datos que probablemente necesita
- Historial de compras y ventas  
- Datos de pago  
- Ubicación de usuarios  
- Actividad dentro de la plataforma  
- Reportes de fraude
