## Análisis del Dataset: Student Performance Factors

Con el dataset **Student Performance Factors**, se podría entrenar un modelo para **predecir la nota final del examen (Exam_Score)** de un estudiante a partir de distintos factores académicos y personales.

Para esto se usaría **aprendizaje supervisado**.  
El aprendizaje supervisado es un tipo de *Machine Learning* en el que el modelo aprende a partir de datos que ya incluyen el resultado correcto. Es decir, el sistema recibe ejemplos con las variables de entrada y la respuesta esperada, y aprende patrones para poder predecir nuevos casos.

En este caso, sería un **problema de regresión**, ya que la variable objetivo (**Exam_Score**) es un número.

Algunas **features importantes** del dataset podrían ser:

- **Hours_Studied**: la cantidad de horas que estudia el estudiante, que probablemente influye directamente en su rendimiento.
- **Attendance**: la asistencia a clases, ya que asistir regularmente suele mejorar la comprensión de los temas.
- **Previous_Scores**: las notas anteriores del estudiante, que pueden indicar su nivel académico previo.
- **Motivation_Level**: el nivel de motivación, que puede afectar cuánto esfuerzo dedica al estudio.
- **Access_to_Resources**: el acceso a recursos de estudio, como libros o internet, que puede facilitar el aprendizaje.

Estas variables ayudan al modelo a **identificar patrones** y entender qué factores influyen más en el resultado final del examen.
